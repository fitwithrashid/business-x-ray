# Business Archetypes

**Pre-built patterns for fast inference during interview.**

Use these to quickly guess business structure, then confirm with user.

---

## How to Use

1. Identify archetype from user's initial description
2. Present pre-filled guess based on archetype
3. User confirms or adjusts
4. Proceed with archetype-informed questions

---

## Archetype 1: Coach / Consultant

### Typical Profile
- **Business Model:** 1-on-1 or group coaching, consulting services
- **Price Range:** $1,500 - $25,000+
- **Team:** Solo or with 1-2 VAs
- **Client Capacity:** 5-20 active clients

### Default Business Map

| Column | Typical Content |
|--------|----------------|
| **TRAFFIC** | YouTube, LinkedIn, Podcast, Referrals |
| **CONVERTERS** | Free training, Assessment, Lead magnet PDF |
| **PRODUCTS** | Discovery call → High-ticket program |
| **FUNNELS** | Email nurture → Application → Call → Enrollment |
| **MATH** | $5K-$25K per client, 2-4 new clients/month |
| **TEAM** | Owner + Admin VA |
| **GOALS** | Scale without losing quality, systematize delivery |

### Default Bow-Tie

- **Awareness:** YouTube videos, LinkedIn posts
- **Nurturing:** Free training, Email list
- **Consideration:** Application form, Discovery call
- **Commit:** Enrollment call, Payment
- **Onboarding:** Intake form, Goals call, Community access
- **Launch:** First coaching session
- **Results:** Transformation, Testimonial, Referral

### Common Pain Points
- Content creation bottleneck (takes too long)
- Discovery calls eating up time
- Onboarding manual and inconsistent
- Follow-up slipping through cracks

## Archetype 2: Course Creator

### Typical Profile
- **Business Model:** Digital courses, memberships, communities
- **Price Range:** $47 - $2,000 for courses, $19-$99/month for memberships
- **Team:** Solo or with 1-2 VAs + contractors
- **Client Capacity:** 10-1000+ students

### Default Business Map

| Column | Typical Content |
|--------|----------------|
| **TRAFFIC** | YouTube, Instagram, Podcasts, Affiliates |
| **CONVERTERS** | Free course, Webinar, Lead magnet |
| **PRODUCTS** | Low-ticket ($47-$297) → Flagship ($997-$2K) → Community |
| **FUNNELS** | Webinar funnel, Email sequence, Upsell sequence |
| **MATH** | High volume, lower touch |
| **TEAM** | Owner + Content VA + Tech support |
| **GOALS** | Evergreen revenue, reduce launch stress |

### Default Bow-Tie

- **Awareness:** YouTube tutorials, Social content
- **Nurturing:** Free mini-course, Email list
- **Consideration:** Webinar, Sales page
- **Commit:** Checkout, Payment plan
- **Onboarding:** Welcome sequence, Course access, Community invite
- **Launch:** First module completion
- **Results:** Course completion, Success stories, Community engagement

### Common Pain Points
- Content creation bottleneck
- Launch burnout (prefer evergreen)
- Low completion rates
- Community management time

## Archetype 3: Agency

### Typical Profile
- **Business Model:** Done-for-you services (marketing, design, development)
- **Price Range:** $2,000 - $20,000/month retainers
- **Team:** Owner + specialists + project manager
- **Client Capacity:** 5-15 active clients

### Default Business Map

| Column | Typical Content |
|--------|----------------|
| **TRAFFIC** | Referrals, LinkedIn, Case studies, Networking |
| **CONVERTERS** | Portfolio, Case studies, Free audit |
| **PRODUCTS** | Project-based ($5K-$50K) or Retainer ($3K-$15K/mo) |
| **FUNNELS** | Referral → Discovery → Proposal → Contract |
| **MATH** | Fewer clients, higher value |
| **TEAM** | Owner + PM + Specialists |
| **GOALS** | Predictable revenue, reduce owner dependency |

### Default Bow-Tie

- **Awareness:** Referrals, LinkedIn, Speaking
- **Nurturing:** Case studies, Newsletter
- **Consideration:** Discovery call, Audit
- **Commit:** Proposal, Contract signed
- **Onboarding:** Kickoff call, Access setup, Process orientation
- **Launch:** First deliverable
- **Results:** Ongoing delivery, Results tracking, Referral request

### Common Pain Points
- Owner doing too much client work
- Proposals taking too long
- Scope creep
- Client communication overhead

## Archetype 4: SaaS / Software

### Typical Profile
- **Business Model:** Software subscription
- **Price Range:** $29 - $299/month
- **Team:** Owner + developers + support
- **Client Capacity:** 100-10,000+ users

### Default Business Map

| Column | Typical Content |
|--------|----------------|
| **TRAFFIC** | Content marketing, SEO, Integrations, AppSumo |
| **CONVERTERS** | Free trial, Freemium tier, Demo |
| **PRODUCTS** | Free → Starter → Pro → Enterprise |
| **FUNNELS** | Sign up → Trial → Upgrade → Retain |
| **MATH** | MRR focus, churn metrics |
| **TEAM** | Owner + Dev + Support + Marketing |
| **GOALS** | Reduce churn, increase LTV, automate support |

### Default Bow-Tie

- **Awareness:** Blog content, SEO, Product Hunt
- **Nurturing:** Free tier, Email onboarding
- **Consideration:** Feature comparison, Demo call (enterprise)
- **Commit:** Upgrade to paid, Annual plan
- **Onboarding:** Product tour, Setup wizard, First use case
- **Launch:** First "aha" moment, Key feature used
- **Results:** Ongoing usage, Expansion, Referral program

### Common Pain Points
- Support tickets overwhelming
- Churn too high
- Onboarding drop-off
- Feature requests backlog

- **Price Range:** $500 - $5,000/month
- **Team:** Solo or with subcontractors
- **Client Capacity:** 5-20 clients

### Default Business Map

| Column | Typical Content |
|--------|----------------|
| **TRAFFIC** | Referrals, Local networking, LinkedIn |
| **CONVERTERS** | Discovery call, Proposal |
| **PRODUCTS** | Monthly retainer, Project packages |
| **FUNNELS** | Referral → Call → Proposal → Start |
| **MATH** | Time-based billing, capacity limited |
| **TEAM** | Owner + subcontractors |
| **GOALS** | Raise prices, systematize delivery, reduce hours |

### Default Bow-Tie

- **Awareness:** Referrals, LinkedIn
- **Nurturing:** Email, Relationship building
- **Consideration:** Discovery call
- **Commit:** Proposal accepted
- **Onboarding:** Intake, Access setup
- **Launch:** First deliverable
- **Results:** Ongoing service, Testimonial, Referral

### Common Pain Points
- Underpriced services
- Scope creep
- No systems, everything in head
- Capacity ceiling

## Using Archetypes in Interview

### Step 1: Detect Archetype
Listen for trigger words in user's business description.

### Step 2: Present Inference
> "Sounds like you're running a [Archetype] business. Based on that, I'm guessing your setup looks something like:
>
> [Show archetype defaults]
>
> How close am I?"

### Step 3: Adjust
Modify defaults based on user corrections.

### Step 4: Flag Gaps
If user's actual setup differs significantly from archetype best practices, flag as potential opportunity.
